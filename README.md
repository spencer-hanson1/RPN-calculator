# RPN-calculator
Written in Python. Since the operator comes after the operands, this algorithm utilizes the stack data structure.

Examples:
* 2 3 + = 5
* 2 4 * = 8
* 2 4 * 6 + = 14

Operators:
* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Power (^)

Commands:
* Summation (sum)
* Rotation (rot)
* Copy (copy)
